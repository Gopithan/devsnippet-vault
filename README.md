Full Stack MERN App to save and manage developer code snippets.

## Tech Stack
- React
- Node.js
- Express
- MongoDB
- JWT Authentication

## Features
- User Register / Login
- Create snippets
- Search & filter snippets
- Edit & delete snippets
- Secure API with JWT

## Run locally

### Backend
cd server  
npm install  
npm run dev  

### Frontend
cd client  
npm install  
npm start
